# Cercador de Pel·lícules Isomòrfic

Aplicació isomòrfica desenvolupada amb JavaScript, Node.js i Express.

## Característiques
- Backend amb Express
- Persistència de dades mitjançant fitxer JSON
- Aplicació isomòrfica amb lògica compartida
- Aproximació CSR
- Desplegada a Railway

## Funcionament
El backend llegeix les dades des d’un fitxer JSON i ofereix una API REST.
El frontend consumeix aquesta API i renderitza les dades al navegador.
